---
title: import
description: Import macros into a local variable.
---

The **import** tag imports [macro](/canvas/tags/macro) names in a local variable. The tag is documented in detail in the documentation for the [macro](/canvas/tags/macro) tag.

```canvas
{% import 'forms.html' as forms %}

<p>{{ forms.input('username') }}</p>
<p>{{ forms.input('password', null, 'password') }}</p>
```
