---
title: cms_list
description: Define an editable list component for the Canopy visual editor.
---

The `cms_list` function is used to define a list component which can be edited on the Canopy editor.

```canvas
{{ cms_list(name, { ordered: ..., items: [ '...', '...' ], class: '...' }) }}
```

## Parameters

<ParamField path="name" type="string" required>
  The element identifier
</ParamField>

<ParamField path="ordered" type="boolean">
  Set ordered list or unordered list
</ParamField>

<ParamField path="items" type="array">
  Array containing the content of each list item
</ParamField>

<ParamField path="class" type="string">
  A space-separated list of CSS classes to style the list component
</ParamField>

<Note>
The `class` variable is only accessible in the code. All other variables are available in the Canopy editor.
</Note>

## List component template

```canvas
<{{ ordered ? 'ol' : 'ul' }} class={{ class }}>
  {% for item in items %}
    <li>{{ item }}</li>
  {% endfor %}
</{{ ordered ? 'ol' : 'ul' }}>
```

## Rendered HTML output

```html
<ol class="bg-pink">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

## Example

```canvas
{{ cms_list('list-tag-123', { ordered: true, items: [ 'Item 1', 'Item 2', 'Item 3' ], class: 'list-decimal pl-5 space-y-2' }) }}
```

```html
<ol class="list-decimal space-y-2 pl-5">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>
```
