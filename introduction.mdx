## Key Features

### Visual Content Editing with Canopy

Canopy allows content editors to update page-specific content like headings, text, images, and buttons directly from the dashboard without touching code.

### Structured Data with Collections

Create reusable, structured data schemas for dynamic content like staff profiles, product listings, testimonials, and more.

### Local Development with Courier

Develop locally with automatic sync, push and pull code effortlessly, and experience the power of Blutui from your preferred development environment.

### Version Control with Cassettes

Manage multiple versions of your website for different seasons, campaigns, or variations without creating entirely new projects.
