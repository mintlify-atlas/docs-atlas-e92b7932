## Introduction

The collections tab can be found in the navigation menu on the right hand side of the Dashboard landing page.

## What are Collections?

Collections are a series of fields that are driving something in a template. They allow you to edit groups of things on your site. For example, a Team collection could contain information about your people. Most websites are collection driven - anything that's a group of information can be a collection.

Think of collections as a database equivalent without the hassle of a database. You can create a data structure and then that structure can be referred to in the template.

Collections are displayed on the left panel, with items in the collection shown on the right. You can edit individual collection items using the edit item button.

Images can be taken from the assets library. You plan out how you want your collection to behave, and depending on what you've set up, you'll be able to edit the individual fields which will then reflect on the front end. You don't have to worry about layout, look or feel because the site will handle that for you.

## Creating a Collection

To create a new collection, click the `+ Create collections` button in the bottom left of the navigation. When you click this you can set a `Name` for the collection. After you give it a name it will create a blank collection.

### Adding Fields to the Collection

You can customise the collection how you like.

### All Field Types

![Collection field types](https://cdn.blutui.com/uploads/assets/Help/Site-Dashboard/Collections/col-2.png)

Here is a list of all field types and what they can do:

| Field         | Description                                      |
| ------------- | ------------------------------------------------ |
| Checkbox      | Select multiple options                          |
| Color         | Color allows you to pick a color and set a color |
| Date          | Set a date dd/mm/yyyy or mm/dd/yyyy              |
| Date & Time   | Set a date and time                              |
| Email         | Set an email address                             |
| File          | Upload files or select files from assets         |
| Number        | Set a number value                               |
| Phone         | Set a phone number                               |
| Radio         | Select one from many options                     |
| Rich Textbox  | Write and manipulate large amount of text        |
| Select        | Set an option from the dropdown                  |
| Text          | Write small amount of text                       |
| Text Area     | Write large amount of text                       |
| Time          | Set a time                                       |
| URL           | Set a URL                                        |

### How to Add Your Collection to the Site

To add this collection to the site you will need access to the code. If this is controlled by the agency who made your site it is recommended to contact them to ask it to be added in the project.

## Adding Items to a Collection

You can add to a collection very simply by hitting the `New entry` button in the top right corner. This will take you to a page which will have fields setup by the agency to fill out.

You might also see under the input some help text which will help provide context to that field.

Some fields might be Required which have to be filled in to add an item to the collection.

![Example of a collection](https://cdn.blutui.com/uploads/assets/Help/Site-Dashboard/Collections/col-3.png)

This is an example of what a collection `New entry` might look like. This may vary from collection to collection.

## Edit a Collection

To edit a collection, you will need to go to the collection you want to edit. For example, the staff collection.

To edit an item in a collection you will want to select the collection you want to edit an item from. This is located on the left side of the screen. You can then click on this to access your collection which will then show a list of all collection items.

![Collection entries](https://cdn.blutui.com/uploads/assets/Help/Site-Dashboard/Collections/col-4.png)

Hit the `Edit` button on the collection entries you want to edit. If you can't find the one you're looking for it might be on another page - use the pagination at the bottom to browse through entries.

This will bring up the same screen as adding an entry but with content inside the fields.
