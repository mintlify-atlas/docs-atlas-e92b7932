## Introduction

The blogs tab can be found second from the top of the navigation menu on the right hand side of the Dashboard landing page.

When landing on the first screen, you will see that there are two options - posts or blogs.

You are also able to see the name of the post, the author, the date it was published and which blog the post is a part of. If you click on the link under 'title,' it will direct you to the blog editor.

## What's the difference between a blog and a post?

A blog is a container for posts. You can create multiple blogs on your site, and each blog can contain multiple posts. Think of a blog as a category or section, and posts as the individual articles within that blog.

## Create a New Blog

With Blutui, you can create as many blogs as you like. In order to create a new blog, all you need to do is click on any plus sign on the page. You can find this sign either next to the 'news' tab or in the bottom left hand corner.

![Blog Creating Blog](https://cdn.blutui.com/uploads/assets/Help/Site-Dashboard/assets/assets-2023-90813-am.png)

## Create a New Post

You can create a post by clicking on the `Create post` button on the top right hand side of the page. You will be put into the blog `editor`.

![Blog Creating Post](https://cdn.blutui.com/uploads/assets/Help/Site-Dashboard/blogs/blog-2023-5.png)

### Adding Content

You can add a `Title`, `Excerpt` and `Content`. You will be able to add all the content into the `Content` section including `images`, `lists` and more.

### Publish Date

You can set a publish time of the post. If you do set this into the future it will not show up on your site.

### Author

You can add a name to who wrote the post here. It will default to the account name your using to write the blog.

### Feature Image

You can upload or search in your `assets` for a featured image for this post.

### Permalink

You can set the url link for the post. If you dont it will generate one automatically.

## Edit a Blog

You can edit a blog by clicking on the `edit` button on the right hand side of the page. You will then be taken to the blog editor where you can edit three main components; the title, description and main copy.

### Editing Functions

You will also be able to use the following functions:

![Blog functions](https://cdn.blutui.com/uploads/assets/Help/Site-Dashboard/blogs/blog-2023-2.png)

1. Normal text and heading formats
2. Bold
3. Italics
4. Underline
5. Strikethough
6. Bullet Point
7. Justification
8. Add a link to a piece of text
9. Insert image
10. Embedded video (vimeo, iframe, YouTube)
11. Blockquote, Code Snippet, Code Block and Divider

The styles of headings, normal text, blockquotes etc. are all controlled by the blog template which you can select when you create the blog. This is a template that your agency will write for you.

## Publish or Delete a Blog

At the top right hand side of the page you will find the blue `Publish` button and the button with the three dots. Clicking on the `Publish` button will allow you to publish your blog post to your site. Please make sure that the timezone of your site is correctly set in the site settings so you know that your blog will publish at the correct date and time for your specific timezone.

If you click on the button with the three dots, this will pull up two options; 1. Delete Post and 2. Back to blogs. When you click `Delete Post`, a confirmation dialogue will come up to confirm your decision. Clicking on `back to blogs` will redirect you to the landing page of the blogs tab.
