## Retrieve current user

Retrieve the currently authenticated user.

<CodeGroup>
```typescript Node.js
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

blutui.user.get()
```

```bash cURL
curl -X GET 'https://api.blutui.com/v1/user' \
     -H 'Authorization: Bearer ey....' \
     -H 'Content-Type: application/json'
```
</CodeGroup>

<ResponseExample>
```json Response
{
  "id": "9c0d7ef9-277e-4360-8fcf-0815853e22a9",
  "object": "user",
  "name": "Luke Skywalker",
  "two_factor_enabled": true,
  "created_at": 1690330767,
  "updated_at": 1703633941
}
```
</ResponseExample>

<ResponseField name="id" type="string">
  The unique identifier for this user.
</ResponseField>

<ResponseField name="object" type="string">
  This will always contain `user` for this endpoint.
</ResponseField>

<ResponseField name="name" type="string">
  The user's full name.
</ResponseField>

<ResponseField name="two_factor_enabled" type="boolean">
  Determine if the user has enabled two factor authentication.
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  The user's created at timestamp.
</ResponseField>

<ResponseField name="updated_at" type="timestamp">
  The user's updated at timestamp.
</ResponseField>

## User Agencies

To access the agencies associated with the authenticated user, use the [Agencies API](/api-reference/agencies/overview):

```typescript
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

// Get current user
const user = await blutui.user.get()

// List all agencies for this user
const agencies = await blutui.agencies.list()
```
