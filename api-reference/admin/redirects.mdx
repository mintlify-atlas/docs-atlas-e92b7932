## List all redirects

<ParamField query="limit" type="integer" default={10}>
  Limit the number of objects to return, ranging between 1 and 20.
</ParamField>

<ParamField query="page" type="integer" default={1}>
  The current page used to display the pagination results.
</ParamField>

<CodeGroup>
```typescript Node.js
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

blutui.project('one').admin.redirects.list()
```

```bash cURL
curl -X GET 'https://{handle}.blutui.com/admin/api/redirects' \
     -H 'Authorization: Bearer ey....' \
     -H 'Content-Type: application/json'
```
</CodeGroup>

<ResponseField name="object" type="string">
  This will always contain `list` for this endpoint.
</ResponseField>

<ResponseField name="data" type="array">
  An array of up to `limit` redirect resources. If no redirects are available, an empty array is returned.
</ResponseField>

<ResponseField name="meta" type="object">
  Additional information about the list.
</ResponseField>

## Create redirect

<api>POST https://{handle}.blutui.com/admin/api/redirects</api>

<ParamField body="from" type="string" required>
  The from url of the redirect.
</ParamField>

<ParamField body="to" type="string" required>
  The to url of the redirect.
</ParamField>

<CodeGroup>
```typescript Node.js
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

blutui.project('one').admin.redirects.create({
  from: '/about',
  to: '/about-us',
})
```

```bash cURL
curl -X POST 'https://{handle}.blutui.com/admin/api/redirects' \
     -H 'Authorization: Bearer ey....' \
     -H 'Content-Type: application/json'
     -d $'{
  "from": "/about",
  "to": "/about-us"
}'
```
</CodeGroup>
