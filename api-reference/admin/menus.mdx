## List all menus

<ParamField query="limit" type="integer" default={10}>
  Limit the number of objects to return, ranging between 1 and 20.
</ParamField>

<ParamField query="page" type="integer" default={1}>
  The current page used to display the pagination results.
</ParamField>

<ParamField query="expand" type="string[]">
  Expand the response for the given items. Expand array accepts `list`.
</ParamField>

<CodeGroup>
```typescript Node.js
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

blutui.project('one').admin.menus.list()
```

```bash cURL
curl -X GET 'https://{handle}.blutui.com/admin/api/menus' \
     -H 'Authorization: Bearer ey....' \
     -H 'Content-Type: application/json'
```
</CodeGroup>

<ResponseField name="object" type="string">
  This will always contain `list` for this endpoint.
</ResponseField>

<ResponseField name="data" type="array">
  An array of up to `limit` menu resources. If no menus are available, an empty array is returned.
</ResponseField>

<ResponseField name="meta" type="object">
  Additional information about the list.
</ResponseField>

## Create menu

**`POST https://{handle}.blutui.com/admin/api/menus`**

<ParamField body="name" type="string" required>
  The name of the menu.
</ParamField>

<ParamField body="handle" type="string" required>
  The unique slug of the menu.
</ParamField>

<ParamField body="items" type="array">
  The list of items in the menu.
</ParamField>

<ParamField body="items[].label" type="string" required>
  The label for the menu item.
</ParamField>

<ParamField body="items[].url" type="string" required>
  The URL for the menu item.
</ParamField>

<ParamField body="items[].active" type="boolean">
  Whether the menu item is active or not.
</ParamField>

<ParamField body="items[].is_new_tab" type="boolean">
  Whether the menu item should open in a new tab or not.
</ParamField>

<CodeGroup>
```typescript Node.js
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

blutui.project('one').admin.menus.create({
  name: 'Primary Menu',
  handle: 'primary-menu',
  items: [
    {
      label: 'About',
      url: '/about',
      isNewTab: false,
      active: true,
    },
  ],
})
```

```bash cURL
curl -X POST 'https://{handle}.blutui.com/admin/api/menus' \
     -H 'Authorization: Bearer ey....' \
     -H 'Content-Type: application/json'
     -d $'{
  "name": "Primary Menu",
  "handle": "primary-menu",
  "items": [
    {
      "label": "About",
      "url": "/about",
      "is_new_tab": false,
      "active": true
    }
  ]
}'
```
</CodeGroup>
