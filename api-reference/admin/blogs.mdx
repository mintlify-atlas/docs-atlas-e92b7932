## List all blogs

<ParamField query="limit" type="integer" default={10}>
  Limit the number of objects to return, ranging between 1 and 20.
</ParamField>

<ParamField query="page" type="integer" default={1}>
  The current page used to display the pagination results.
</ParamField>

<ParamField query="expand" type="string[]">
  Expand the response for the given items. Expand array accepts `list`.
</ParamField>

<CodeGroup>
```typescript Node.js
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

blutui.project('one').admin.blogs.list()
```

```bash cURL
curl -X GET 'https://{handle}.blutui.com/admin/api/blogs' \
     -H 'Authorization: Bearer ey....' \
     -H 'Content-Type: application/json'
```
</CodeGroup>

<ResponseField name="object" type="string">
  This will always contain `list` for this endpoint.
</ResponseField>

<ResponseField name="data" type="array">
  An array of up to `limit` blog resources. If no blogs are available, an empty array is returned.
</ResponseField>

<ResponseField name="meta" type="object">
  Additional information about the list.
</ResponseField>

## Create blog

**`POST https://{handle}.blutui.com/admin/api/blogs`**

<ParamField body="name" type="string" required>
  The name of the blog.
</ParamField>

<ParamField body="uri" type="string" required>
  The URL used to view all posts that belong to this blog.
</ParamField>

<ParamField body="slug" type="string">
  The unique slug of the blog.
</ParamField>

<ParamField body="layout" type="string">
  The layout used to display all blog posts.
</ParamField>

<ParamField body="post_layout" type="string">
  The layout used to display a single post in this blog.
</ParamField>

<CodeGroup>
```typescript Node.js
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

blutui.project('one').admin.blogs.create({
  name: 'Blog',
  uri: '/blog',
})
```

```bash cURL
curl -X POST 'https://{handle}.blutui.com/admin/api/blogs' \
     -H 'Authorization: Bearer ey....' \
     -H 'Content-Type: application/json'
     -d $'{
  "name": "Blog",
  "uri": "/blog"
}'
```
</CodeGroup>

```json Response
}
```
</ResponseExample>

<ResponseField name="object" type="string">

<ResponseField name="name" type="string">
  The name of this blog.
</ResponseField>

<ResponseField name="slug" type="string">
  The unique slug for this blog.
</ResponseField>

<ResponseField name="uri" type="string">
  The URL used to view all posts that belong to this blog.
</ResponseField>

<ResponseField name="layout" type="string">
