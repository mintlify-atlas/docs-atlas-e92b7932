## List all form submissions

<ParamField path="form_id" type="string" required>
  The ID of the form.
</ParamField>

<ParamField query="limit" type="integer" default={10}>
  Limit the number of objects to return, ranging between 1 and 20.
</ParamField>

<ParamField query="page" type="integer" default={1}>
  The current page used to display the pagination results.
</ParamField>

<ParamField query="expand" type="string[]">
  Expand the response for the given items. Expand array accepts `list`.
</ParamField>

<CodeGroup>
```typescript Node.js
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

blutui.project('one').admin.submissions.list('form-id')
```

```bash cURL
curl -X GET 'https://{handle}.blutui.com/admin/api/forms/form-id/submissions' \
     -H 'Authorization: Bearer ey....' \
     -H 'Content-Type: application/json'
```
</CodeGroup>

<ResponseField name="object" type="string">
  This will always contain `list` for this endpoint.
</ResponseField>

<ResponseField name="data" type="array">
  An array of up to `limit` form submission resources. If no form submissions are available, an empty array is returned.
</ResponseField>

<ResponseField name="meta" type="object">
  Additional information about the list.
</ResponseField>

## Retrieve form submission

<ParamField path="form_id" type="string" required>
  The ID of the form.
</ParamField>

<ParamField path="id" type="string" required>
  The ID of the submission to retrieve.
</ParamField>

<CodeGroup>
