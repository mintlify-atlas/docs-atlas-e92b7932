## Retrieve cassette

**`GET https://api.blutui.com/v1/agencies/{username}/cassettes/{id}`**

<ParamField path="username" type="string" required>
  The agency username.
</ParamField>

<ParamField path="id" type="string" required>
  The ID of the cassette to retrieve.
</ParamField>

<CodeGroup>
```typescript Node.js
import Blutui from 'blutui'

const blutui = new Blutui('ey....')

blutui.agency('studio').cassettes.get('cassette-id')
```

```bash cURL
curl -X GET 'https://api.blutui.com/v1/agencies/studio/cassettes/cassette-id' \
     -H 'Authorization: Bearer ey....' \
     -H 'Content-Type: application/json'
```
</CodeGroup>
